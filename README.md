# 人力资源管理系统 java770

## 项目概述

人力资源管理系统是基于SpringBoot和Vue技术栈的企业级应用。该系统致力于提升企业人力资源管理的效率，通过信息化手段对员工信息、考勤、工资等活动进行整合和管理。

## 技术栈

- **前端技术：**
  - Html
  - Css
  - Js
  - Vue
  - Element-ui
- **数据库：**
  - MySQL
- **后端技术：**
  - Java
  - Spring Boot
  - MyBatis

## 运行环境

- **开发工具：**
  - IDEA
  - Eclipse
- **数据库：**
  - MySQL 8.0
- **数据库管理工具：**
  - Navicat 10及以上版本
- **环境配置软件：**
  - JDK 1.8
  - Maven 3.6.3
- **前端Nodejs版本：**
  - Nodejs 16

## 功能模块

系统包含以下五大核心功能模块：

1. **首页**
   - 系统统计信息展示，包括：
     - 员工情况
     - 登录用户信息
     - 考勤记录
     - 社保缴纳比例
     - 员工入职情况
     - 员工分布情况
   - 图表展示（折线图、柱状图、饼图）
2. **薪资管理**
   - 五险一金计算
   - 参保城市设置
   - 工资管理
3. **权限管理**
   - 角色管理
   - 菜单管理
4. **系统管理**
   - 文件管理
   - 员工管理
   - 部门管理
5. **考勤管理**
   - 请假审批
   - 考勤表现

## 系统角色

系统包含但不限于以下角色：
- 系统管理员
- 薪资管理员
- 权限管理员
- 员工
- 部门经理
